# Introduction-Machine Learning for Graphs

# 不同类型的机器学习任务(Different types of tasks)

  • 节点级别任务Node-level

    • 蛋白质折叠（Protein Folding）
      • 关键点：空间图（Spatial graph）
      • Nodes：蛋白质序列中的氨基酸
      • Edges：氨基酸(残基)之间的邻近性
      • 仅仅基于蛋白质的氨基酸序列就可以计算出蛋白质的三维结构

  • 边缘级Edge-level

    • 推荐系统（Recommender Systems）
      • Nodes：用户 和 商品
      • Edges：用户和购买的商品关系
      • Goal：推荐用户可能会喜欢的商品
    • 药物副作用预测（Drug Side Effects）
      • Nodes：药物 和 蛋白质
      • Edges：医疗作用
      • Goal：给两种药物预测副作用

  • 团(子图)级Community（subgraph）-level

    • 交通预测（Traffic Prediction）
      • Nodes：公路段
      • Edges：路段之间的连通性

  • 图级别Graph-level、图生成Graph generation

    • 药物发现（Drug Discovery）

      • Nodes: 原子（Atoms）
      • Edges：化学键（Chemical bonds）

    • 分子生成和优化（Molecule generation/Optimization）

      • 产生与药物高度相似的新分子
      • 优化现有分子，使其具有理想的性质

    • 图形演化（Physical simulation）

      • Nodes：粒子（ Particles）
      • Edges：粒子之间的相互作用
      • Goal：预测图将如何演化

# 图表征的选择（Choice of Graph Representation）

  • 网络组成(Components of a Network)：
    • Objects: N ​-> nodes or vertices
    • Interactions: E -> links or edges
    • System: G(N,E) -> network or graph
  • 图的分类
    • 有向图(Directed) and 无向图(Undirected)
      • 举例：朋友圈是无向图 / Github 中的Follow 有向图
    • 二分图(Bipartite graph)
      • 举例：Authors-to-Papers / Actors-to-Movies
    • 有权图(Weighted ) and 无权图(Unweighted )
    • 自环图(Self-edges (self-loops))
    • 多重图(Multigraph)
