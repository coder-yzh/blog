# 1.2.2 Node centrality（节点中心）

  • Node centrality c takes the node importance in a graph into account-考虑节点在图中的重要性
  • 建立重要性模型的不同方法:
    • Engienvector(特征向量) centrality
      • A node is important if surrounded by important neighboring nodes u∈N(v)
      • the sum of the centrality of neighboring nodes
    • Betweenness(间隙) centrality
      • A node is important if it lies on many shortest paths between other nodes.
    • Closeness(紧密性) centrality
      • A node is important if it has small shortest path lengths to all other nodes.

# 1.2.3 Clustering coefficient（聚类系数）

# 1.2.4 Graphlets

  • Rooted connected non-isomorphic subgraphs(根连通非同构子图)
  • Graphlet Degree Vector (GDV)：表征节点V的邻里结构的向量，提供了一个节点的局部网络拓扑的度量

# 1.2.5 Summary

1、Goal：Characterize the structure and position of a node in the network

2、不同特征的计算角度不同：

  • Degree counts #(edges) that a node touches
  • Clustering coefficient counts #(triangles) that a node touches.
  • GDV counts #(graphlets) that a node touches

3、节点特征的获取方法可归类为：

  • Importance-based features
    • Node degree
      • Simply counts the number of neighboring nodes
    • Different node centrality measures
      • Models importance of neighboring nodes in a graph
      • Different modeling choices: eigenvector centrality, betweenness centrality, closeness centrality
    • 实际应用：predicting celebrity users in a social network（预测社交网络中的名人用户）
  • Structure-based features
    • Node degree
      • Counts the number of neighboring nodes
    • Clustering coefficient
      • Measures how connected neighboring nodes are
    • Graphlet count vector
      • Counts the occurrences of different graphlets
    • 实际应用：Predicting protein functionality in a protein-protein interaction network（预测蛋白质相互作用网络中的蛋白质功能）

# 3.1 Graph-level Tasks

Goal: We want features that characterize the structure of an entire graph.

# 3.2 Background: Kernel Methods

  • Kernel methods(核方法)：widely-used for traditional ML for graph-level prediction
  • Idea: Design kernels instead of feature vectors

# 3.2 Graph-level Features

# 3.2.1 Graph Kernels

1、Measure similarity between two graphs，即测量两个图之间的相似性

  • Graphlet Kernel
  • Weisfeiler-Lehman Kernel
  • Random-walk kernel
  • Shortest-path graph kernel

2、Graph Kernels-核心思想

  • Goal: Design graph feature vector φ(G)
  • Key idea: Bag-of-* for a graph
    • Bag-of-Words (BoW)
      • Regard nodes as words,以节点为词，G为节点个数
    • Bag of node degrees
      • G为节点度数向量
    • Graphlet Kernel and Weisfeiler-Lehman (WL) Kernel use Bag-of-* representation of graph，where *** is more sophisticated** than node degrees!
