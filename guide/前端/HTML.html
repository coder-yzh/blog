    # HTML内容摘要

    # 1、网页的基本元素

    # h元素

    • h元素有助于网站的SEO优化，可以促进关键字排名
    • 建议在网页中最多只有1个h1标签
    • 乱用h元素不仅不会给网站带来好的权重，同时也有可能被搜索引擎认为作弊，最后导致k站

    # span元素

    • 默认情况下，和普通文本没有区别
    • 增加特殊样式，主要作用是对普通文本进行归类

    # div元素

    • 把网页分割成多个独立的部分
    • 一般作为其他元素的父容器，把其他元素包住，代表一个整体

    # img元素

    • 单标签，专门用于显示图片
    • 支持的图片格式：
      • png：静态图片，支持透明
      • jpg：静态图片，不支持透明
      • gif：动态图片
    • alt属性：当图片失效（加载失败时）显示文本
    • src属性：一般为相对路径
    • width属性、height属性的默认单位是像素（px）

    # a元素

    • 定义超链接，打开新的URL
    • href属性（Hypertext Reference）：指定要打开的URL
    • target属性：如何打开URL
      • _self:默认值,在当前窗口打开URL
      • _blank:在一个新的窗口中打开URL
      • _parent:在父窗口中打开URL
      • _top:在顶层窗口中打开URL
      • 某个frame的name值:在某个frame中打开URL
    <body>
    <!-- a元素的基本使用 -->
    <a href="http://www.baidu.com">百度一下</a>
    
    <!-- target:
        _self（默认值）:自己
        _blank(会打开新窗口)：空白
    -->
    <div>
        <a href="https://www.baidu.com" target="_blank">百度一下</a>
    </div>
</body>

    # HTML文档声明

    <!DOCTYPE html>
    1
    • 让浏览器用HTML5的标准去解析识别HTML文档
    • 必须放在HTML文档的前面，不省省略，不然会出现兼容性问题

    # HTML元素

    • html元素是HTML文档的根元素，一个文档只能又一个，其他所有元素的都是它的后代元素

    • W3C标准建议为html元素增加一个lang属性，作用是：

      帮助翻译工具来确定要使用的翻译准则

      帮助语音合成工具确定要使用的发音

    有些网站加了lang，一般中文网站中lang="zh-CN"，国外网站设置为lang="en"

    lang->language(语言)

    3、meta元素
