# 三数之和 (opens new window)

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素a，b，c ，使得 *a + b + c =* 0 ？请你找出所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

示例 1：

输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
1
2

示例 2：

输入：nums = []
输出：[]
1
2

示例 3：

输入：nums = [0]
输出：[]
1
2

提示：

  • 0 <= nums.length <= 3000
  • -105 <= nums[i] <= 105

AC代码

class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums = sorted(nums)  # 等同于 nums.sort()
        ans = []
        k = 0
        while k < len(nums) - 2:
            if nums[k] > 0: break
            if k > 0 and nums[k] == nums[k - 1]:
                k += 1
                continue
            i = k + 1
            j = len(nums) - 1
            while i < j:
                dig = nums[k] + nums[i] + nums[j]
                if dig == 0:
                    ans.append([nums[k], nums[i], nums[j]])
                    # print("{0}+{1}+{2},i:{3},j:{4},k:{5}".format(nums[k], nums[i], nums[j], i, j, k))
                    j -= 1
                    i += 1
                    while nums[j] == nums[j + 1] and i < j:
                        j -= 1
                    while nums[i] == nums[i - 1] and i < j:
                        i += 1
                elif dig > 0:
                    j -= 1
                    while nums[j] == nums[j + 1] and i < j:
                        j -= 1
                elif dig < 0:
                    i += 1
                    while nums[i] == nums[i - 1] and i < j:
                        i += 1
            k += 1
        return ans

  • 双指针法铺垫： 先将给定 nums 排序，复杂度为 O(NlogN)。
  • 双指针法思路： 固定 3 个指针中最左（最小）数字的指针 k，双指针 i，j 分设在数组索引 (k, len(nums)) 两端，通过双指针交替向中间移动，记录对于每个固定指针 k 的所有满足 nums[k] + nums[i] + nums[j] == 0 的 i,j 组合：
    • 当 nums[k] > 0 时直接break跳出：因为 nums[j] >= nums[i] >= nums[k] > 0，即 3 个数字都大于0 ，在此固定指针 k 之后不可能再找到结果了。
    • 当 k > 0且nums[k] == nums[k - 1]时即跳过此元素nums[k]：因为已经将 nums[k - 1] 的所有组合加入到结果中，本次双指针搜索只会得到重复组合。
    • i，j 分设在数组索引 (k, len(nums)) 两端，当i < j时循环计算dig = nums[k] + nums[i] + nums[j]，并按照以下规则执行双指针移动：
      • 当dig < 0时，i += 1并跳过所有重复的nums[i]；
      • 当dig > 0时，j -= 1并跳过所有重复的nums[j]；
      • 当dig == 0时，记录组合[nums[k], nums[i], nums[j]]至ans，执行i += 1和j -= 1并跳过所有重复的nums[i]和nums[j]，防止记录到重复组合。

# 最接近的三数之和 (opens new window)

给定一个包括 n 个整数的数组 nums 和 一个目标值 target。找出 nums 中的三个整数，使得它们的和与 target 最接近。返回这三个数的和。假定每组输入只存在唯一答案。

示例：

输入：nums = [-1,2,1,-4], target = 1
输出：2
解释：与 target 最接近的和是 2 (-1 + 2 + 1 = 2) 。
1
2
3

提示：

  • 3 <= nums.length <= 10^3
  • -10^3 <= nums[i] <= 10^3
  • -10^4 <= target <= 10^4

AC代码

class Solution:
    def threeSumClosest(self, nums: List[int], target: int) -> int:
        le = len(nums)
        nums.sort()
        # print(nums)
        num = 100000
        ans = 0
        for k in range(le - 2):
            if k > 0 and nums[k] == nums[k - 1]: continue
            i, j = k + 1, le - 1
            while i < j:
                dig = nums[k] + nums[i] + nums[j] - target
                if dig > 0:
                    if num > dig:
                        num = min(num, dig)
                        ans = nums[k] + nums[i] + nums[j]
                    j -= 1
                    while i < j and nums[j] == nums[j + 1]:
                        j -= 1
                elif dig < 0:
                    if num > abs(dig):
                        num = min(num, abs(dig))
                        ans = nums[k] + nums[i] + nums[j]
                    # print("*********nums[k]:{0},nums[i]:{1},nums[j]:{2},ans:{3}".format(nums[k], nums[i], nums[j], ans))
                    i += 1
                    while i < j and nums[i] == nums[i - 1]:
                        i += 1
                elif dig == 0:
                    return target
                # print("nums[k]:{0},nums[i]:{1},nums[j]:{2},ans:{3}".format(nums[k], nums[i], nums[j], ans))
        return ans

# 四数之和 (opens new window)

给你一个由 n 个整数组成的数组 nums ，和一个目标值 target 。请你找出并返回满足下述全部条件且不重复的四元组 [nums[a], nums[b], nums[c], nums[d]] ：

  • 0 <= a, b, c, d < n
  • a、b、c 和 d 互不相同
  • nums[a] + nums[b] + nums[c] + nums[d] == target

你可以按 任意顺序 返回答案 。

示例 1：

输入：nums = [1,0,-1,0,-2,2], target = 0
输出：[[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]
1
2

示例 2：

输入：nums = [2,2,2,2,2], target = 8
输出：[[2,2,2,2]]
1
2

提示：

  • 1 <= nums.length <= 200
  • -109 <= nums[i] <= 109
  • -109 <= target <= 109

AC代码

class Solution:
    def fourSum(self, nums: List[int], target: int) -> List[List[int]]:
        nums.sort()
        # print(nums)
        le = len(nums)
        ans = []
        for x in range(0, le - 3):
            if x > 0 and nums[x] == nums[x - 1]: continue
            for k in range(x + 1, le - 2):
                if k > x + 1 and nums[k] == nums[k - 1]: continue
                i, j = k + 1, le - 1
                while i < j:
                    dig = nums[x] + nums[i] + nums[j] + nums[k]
                    if dig > target:
                        j -= 1
                        while nums[j] == nums[j + 1] and i < j:
                            j -= 1
                    elif dig < target:
                        i += 1
                        while nums[i] == nums[i - 1] and i < j:
                            i += 1
                    else:
                        ans.append([nums[x], nums[i], nums[j], nums[k]])
                        i += 1
                        j -= 1
                        while nums[i] == nums[i - 1] and i < j:
                            i += 1
                        while nums[j] == nums[j + 1] and i < j:
                            j -= 1
        return ans
