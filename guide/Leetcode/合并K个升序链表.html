# 合并K个升序链表 (opens new window)

给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

示例 1：

输入：lists = [[1,4,5],[1,3,4],[2,6]]
输出：[1,1,2,3,4,4,5,6]
解释：链表数组如下：
[
  1->4->5,
  1->3->4,
  2->6
]
将它们合并到一个有序链表中得到。
1->1->2->3->4->4->5->6

提示：

  • k == lists.length
  • 0 <= k <= 10^4
  • 0 <= lists[i].length <= 500
  • -10^4 <= lists[i][j] <= 10^4
  • lists[i] 按 升序 排列
  • lists[i].length 的总和不超过 10^4

AC代码

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeKLists(self, lists: List[ListNode]) -> ListNode:
        if len(lists)==0:
            lists.append([])
        elif len(lists)==1:
            return lists[0]
        prehead=ListNode(-1)
        prev=prehead
        l1=lists[0]
        for i in range(1,len(lists)):
            l2=lists[i]
            while l1 and l2:
                if l1.val<=l2.val:
                    prev.next=l1
                    l1=l1.next
                else:
                    prev.next=l2
                    l2=l2.next
                prev=prev.next
            prev.next=l1 if l1 is not None else l2
            l1=prehead.next
            prev=prehead
        return prehead.next
