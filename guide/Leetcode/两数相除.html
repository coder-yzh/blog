# 两数相除 (opens new window)

给定两个整数，被除数 dividend 和除数 divisor。将两数相除，要求不使用乘法、除法和 mod 运算符。

返回被除数 dividend 除以除数 divisor 得到的商。

整数除法的结果应当截去（truncate）其小数部分，例如：truncate(8.345) = 8 以及 truncate(-2.7335) = -2

示例 1:

输入: dividend = 10, divisor = 3
输出: 3
解释: 10/3 = truncate(3.33333..) = truncate(3) = 3

输入: dividend = 7, divisor = -3

提示：

  • 被除数和除数均为 32 位有符号整数。
  • 除数不为 0。
  • 假设我们的环境只能存储 32 位有符号整数，其数值范围是 [−2^31, 2^31 − 1]。本题中，如果除法结果溢出，则返回 2^31 − 1。

AC代码

class Solution:
    def divide(self, dividend: int, divisor: int) -> int:
        maxn=(2<<30)-1
        minn=-2<<30
        f=1
        if dividend<0:
            f*=-1
            dividend=-dividend
        if divisor<0:
            f*=-1
            divisor=-divisor
        dividend=str(dividend)
        i=0
        s=[]
        num=0
        while i<len(dividend):
            num=num*10+int(dividend[i])
            k=0
            while num>=divisor:
                k+=1
                num-=divisor
                if num<divisor:
                    break
            i+=1
            s.append(k)
        ans=0
        for i in s:
            ans=ans*10+i
        ans=ans*f
        if ans>maxn or ans<minn:
            return maxn
        return ans

优化代码

class Solution:
    def divide(self, dividend: int, divisor: int) -> int:
        maxn = (2 << 30) - 1
        minn = -2 << 30
        f = 1
        if dividend < 0:
            f = -f
            dividend = -dividend
        if divisor < 0:
            f = -f
            divisor = -divisor

        def div(dividend, divisor):
            if dividend < divisor:
                return 0
            a, b, c = dividend, divisor, 1
            while b + b < a:
                b += b
                c += c
            return c + div(dividend - b, divisor)

        ans = div(dividend, divisor) if f > 0 else -div(dividend, divisor)
        if ans > maxn or ans < minn:
            return maxn
        return ans
