# ４、字典

字典（dictionaries） 是包含键值对（key-value）的python对象。在下面的字典示例中，键 是“name”和“eye_color”变量。它们各自都有与之相关的值。 字典不能有两个相同的键。

例４.１

# 创建字典
goku = {"name": "Goku",

print (goku)
print (goku["name"])
print (goku["eye_color"])

# ６、循环语句

    x += 1 # 等价于 x = x + 1
    print ("i={0}, x={1}".format(i, x)) # 打印多个变量

# ７、函数

函数（functions）是模块化可重用代码块的一种方法。

    x += 2
    return x

    return joined_name

# ８、类

类（classes） 是面向对象的Python编程的基础部分。

例８.１

# 创建函数
class Pets(object):
  
    # 类初始化
    def __init__(self, species, color, name):
        self.species = species
        self.color = color
        self.name = name

    # 用于打印
    def __str__(self):
        return "{0} {1} named {2}.".format(self.color, self.species, self.name)

    # 示例函数
    def change_name(self, new_name):
        self.name = new_name
