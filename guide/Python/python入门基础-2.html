# 2.3位运算

'''
位运算：
&   |   ^   ~   >>  <<
&   and
|   or
^   异或  相同为假，不同为真
~   not
'''

n1 = 0b0110  # 6

n2 = 0b0010  # 2

print(n1 & n2)

print('-' * 20)

print(n1 | n2)

print(n1 ^ n2)

print(~n1)
'''
-6    二进制的负数表示，保存的是补码
6的原码为0000 0110，在计算机中存储形式（补码）为0000 0110
~6运行以后得到二进制1111 1001，这是补码存储形式，补码减一取反，同时符号位不变得到其原码为1000 0111，所以输出为-7

练习
打印出的是十进制
1、print(~7)   ->  0000 0111   ->  1111 1000   ->  1111 0111   ->  1000 1000   ->  -8
2、-9的二进制表示:   原码->1000 1001 补码->1111 0111
3、print(~-4)    ->  -4  ->  1000 0100   ->  1000 0011   ->  1111 1100   ->  0000 0011 ->    3
4、1111 1101的十进制表示：1111 1100 ->  1000 0011   ->  -3
'''
print(~7)
print(bin(-3))
print(~-4)


n = 12  # 0000 1100
print(n << 1)  # 24 12*2
print(n << 2)  # 48 12*2*2
print(n << 3)  # 96 12*2*2*2

# 右移得到的是整除结果
print(n >> 1)  # 6 12/2
print(n >> 2)  # 3 12/(2*2)
print(n >> 3)  # 1 12/(2*2*2)

'''
n=89    右移5位    结果：89/32=2
n=93    左移3位    结果:93*8=744
'''

print('*' * 50)

'''
**  优先级最高

'''
x = 20
result = 20 * 5 ** 2
print(result)

a = 1
b = 2
print(b > a + x < a + b)  # 2>21<3 False

print(b > a + (x < a + b))  # 2>1+0 True

print(a + True)  # 2 False=0,True=1

'''
总结：
1、算术运算符
2、赋值运算符 = += -=
3、比较运算符 > < >= <= == != ...
4、逻辑运算符 and or not
5、位运算   & | ^ ~ << >> -7负数的二进制
6、运算符的优先级:()
'''
