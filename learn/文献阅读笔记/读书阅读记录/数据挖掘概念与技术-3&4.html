# 数据挖掘概念与技术-3&4

# 三、数据预处理

# 1、相关概念

  • 数据预处理的主要步骤：数据清理、数据集成、数据规约和数据变换
    • 数据清理（data cleaning）：通过填写缺失的值，光滑噪声数据，识别或删除离群点，并解决不一致性来“清理”数据
    • 数据集成（data integration）：将多个数据源（例如数据库、数据立方体或文件）中的数据合并，存放在一个一致的数据存储中，如存放在数据仓库中
    • 数据归约（data reduction）：得到数据集的简化表示，数据归约策略包括维归约和数值规约
      • 维归约：使用数据编码方案，以便得到原始数据的简化或者“压缩”表示。例如数据压缩技术（小波变换、主成分分析）、属性子集选择（去掉不相关的属性）和属性构造（从原来的属性集导出更有用的小属性集）
      • 数值规约：使用参数模型（回归和对数线性模型）或者非参数模型（直方图、聚类、抽样或数据聚类），用较小的表示取代数据
    • 数据变换（data transformation）：通过规范化、数据离散化和概念分层产生都是某种形式的数据变换
    • 注：冗余数据的删除既是一种数据清理形式，也是一种数据规约

# 2、数据清理

2.1缺失值的处理方法

  • 忽略元组：当缺少类标号时通常这么做（假定挖掘任务涉及分类）
  • 人工填写缺失值：费时，数据集很大、缺失很多值时，不可行
  • 使用一个全局常量填充缺失值
  • 使用属性的中心度量（如均值或中位数）填充缺失值：对于对称的数据分布而言，可以使用均值，倾斜数据分布应该使用中位数
  • 使用与给定元组属同一类的所有样本的属性均值或中位数
  • 使用最可能的值填充缺失值：可以用回归、使用贝叶斯形式化方法的基于推理的工具或决策树归纳确定

2.2数据光滑技术

  • 噪声：被测量的变量的随机误差或方差
  • 分箱：分箱方法通过考察数据的“近邻”来光滑有序数据值
    • 用箱均值光滑：箱中每一个值都被替换为箱中的均值
    • 用箱中位数光滑：箱中每一个值都被替换为箱中的中位数
    • 用箱边界光滑：给定箱中的最大和最小值同样被视作边界值，箱中的每一个值都被替换为最近的边界值
  • 回归：用一个函数拟合数据来光滑数据。分为线性回归和多元线性回归
  • 离群点分析：通过如聚类来检测离群点。聚类将类似的值组织成群或“簇”。落在簇集合之外的值被视为离群点
  • 注：许多数据光滑的方法也用于数据离散化（一种数据变换）和数据规约

# 3、数据集成

  • 数据集成需要解决实体识别问题和冗余，其中冗余可以被相关分析检测到。对于标称数据，使用卡方检测；对于数值属性，使用相关系数（又称为Pearson积矩系数）和协方差。
  • 数据集成需要解决元组重复问题
  • 数据集成涉及数据值冲突的检测与处理

# 4、数据规约

  • 数据规约策略包括维归约、数量规约和数据压缩
    • 维归约（dimensionality reduction）：减少所考虑的随机变量或属性的个数。包括小波变换和主成分分析，这些方法把原数据变换或投影到较小的空间。属性子集选择是一种维归约方法，其中不相关、弱相关或冗余的属性或维被检测和删除
    • 数量规约（numerosity reduction）：用替代的、较小的数据表示形式替换原数据。有参数方法（回归和对数线性模型）或者非参数方法（直方图、聚类、抽样或数据聚类）
    • 数据压缩（data compression）：使用变换，以便得到原数据的规约或“压缩”表示。
      • 如果原数据能够从压缩后的数据重构，而不损失信息，则该数据规约称为无损的
      • 如果原数据能够近似重构原始数据，则该数据规约称为有损的
    • 注：维归约和数量规约也可以视为某种形式的数据压缩

# 5、数据变换与数据离散化

  • 数据变换策略
    • 光滑：去掉数据中的噪声。这类技术包括分箱、回归和聚类
    • 属性构造（或特征构造）：可以由给定的属性构造新的属性并添加到属性集中，以帮助挖掘过程
    • 聚类：对数据进行汇总或聚集
    • 规范化：把属性数据按比例缩放，使之落入一个特定的小区间
    • 离散化：数值属性的原始值用区间标签或概念标签替换
    • 由标称数据产生概念分层：许多标称属性的概念分层都蕴含在数据库的模式中，可以在模式定义级自动定义
  • 数据规范化
    • 最小-最大规范化：对原始数据进行线性变换，这种变换方法保持原始数据值之间的联系
    • z分数（z-score）规范化：又称为零均值规范化，当属性A的实际最小值和最大值未知，或离群点左右了最小-最大规范化时，此方法有效
    • 按小数定标规范化：通过移动属性A的值的小数点位置进行规范化
