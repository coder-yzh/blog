# 2021年秋季课程目录

# 应用数学基础

  • 闭卷考试，一锤定音，考多少分就是多少分，无平时分。

# 习中特

  • 闭卷考试*70%+课堂表现与专题研讨*20%+出勤*10%
    • 8个专题，每章一题，简答题
  • 教学形式
    • 第一阶段前8周（第7,9-15周）：课堂讲授，每周一次，第二阶段
    • 第二阶段即最后一周（第16周）：组织专题研讨
  • 第16周研讨内容
    • 时间：第16周
    • 内容：围绕习近平新时代中国特色社会主义思想重大理论与现实问题，内容自定
    • 形式：要求人人参加，研讨方式如下：建议分成6-8个小组，每组确定一个主题进行集体备课，最后确定1名代表发言，时间15-20分钟左右。第13周前报研讨主题，第15周前交PPT
    • 课程论文：每人提交一篇2000字的课程学习心得体会
  • 老师推荐的第一本书好好看看+主要ppt
  • 推荐书籍：
    • 中共中央宣传部，习近平新时代中国特色社会主义思想学习纲要
    • 中共中央宣传部，习近平新时代中国特色社会主义思想学习问答
    • 中共中央党校，习近平新时代中国特色社会主义思想基本问题
  • 考试时间：2021-12-17

# 深度学习

  • 考察方式
    • 课程实验：完成基础代码和拓展代码，占50%
    • 研究报告+汇报PPT：选择某一方向归纳现状、挑战、未来方向等，并撰写研究报告，占30%
    • 课堂表现：课程签到，参与程度，占20%

# 先进数据挖掘技术

  • 考查方式
    • 中国大学MOOC学习成绩20%，拿到认证证书
    • 智慧树网学习成绩20%
    • Educoder实践平台成绩20%
    • 互联网竞赛25%
      • 排名成绩20%
      • 提交PPT文档：PPT质量5%（题目来源0.5%，题目内容0.5%，题目数据0.5%，求解思路1%，详细过程1%，实验结果1%，排名结果截图0.5%）
    • 课程文字稿撰写训练10%
      • 要求：将PPT内容进行文字整理，形成通顺的文字
      • 通顺性、美观性
    • 课堂回答问题表现5%

# 数据分析与建模

  • 考核方式：（2-3人组）研究报告+课堂报告+过程演示
  • 考核标准：课程表现20%+课后作业30%+大作业50%
  • 考核成果形式和内容要求
    • 选题应符合教学内容框架，鼓励与个人研究方向结合，应用课堂讲授知识，最好有深化或扩展
    • 选题背景不限，但应有实际意义，面向装备、工程、信息等领域的实际问题，不是教学习题；应该通过数据分析与建模给出启示性(insight)，而非明显结论
    • 完成形式：报告Word/PDF文件（按学术论文规范）、数据及代码、汇报PPT（10分钟），演示视频，报告封面注明小组成员及分工
    • 课堂报告：每组汇报+演示/展示10分钟，质疑和讨论5分钟，重点介绍：案例背景、数据分析与建模过程、针对分析结果的讨论、创新点（如果有的话，包括方法理论、新规律新现象的发现）、研究展望
  • 提示：
    • 研究报告内容须符合课堂讲授内容

# 信息检索

  • 考核方式
    • 自选主题进行信息检索，提交检索报告(30分)+综述报告(30分)+随堂作业(40分)
    • 章节测验+课后作业+签到+讨论
    • 检索报告：1000字左右
    • 综述报告：正文4000字左右（参考文献不计入字数）、格式按照拟设稿期刊排版
  • 检索报告模板
    • 选题
      • 拟定选题名称
      • 选题来源（项目或工作选题、导师选题或兴趣选题）
      • 选题理由及目的
    • 信息检索实施过程（图书馆或互联网，关键检索步骤可截图显示）
      • 检索工具名称，简要说明选择理由
      • 记录检索信息的过程，包括数据库名称、检索式及命中数、分析每次检索结果的相关性和调整过程
      • 简要说明在信息检索过程中的经验教训
    • 使用文献管理软件管理资料
      • 建立专题库，筛选文献，就选题提供有用的信息列表。从文献管理软件中按照投稿论文格式导出主要参考文献20条
    • 综述简介（拟撰写综述的内容）
    • 拟投稿期刊（拟投稿的原因）
  • 综述报告模板
    • 标题
    • 作者、作者单位、邮编
    • 中英文摘要及关键词
    • 正文
    • 参考文献（近5年参考文献在5条以上，核心期刊参考文献在80%以上）
  • 提交要求
    • 检索报告及综述报告电子版（PDF格式）、维普查重报告PDF（只需要原文对照报告的PDF），请于11月20日之前提交至邮箱xinxjs@126.com
    • 文件夹命名规则：学号+姓名+联系电话
    • 文件命名规则
      • XS21023021+杨宗浩+检索报告
      • XS21023021+杨宗浩+综述报告
      • XS21023021+杨宗浩+查重报告
    • 请以压缩文件夹作为附件

# 模式识别

  • 成绩评定：平时成绩+公开测试
  • 公开测试决定成绩的级别A还是B
  • 平时表现决定了成绩的档次是+还是-
  • 平时表现由课堂作业和网课成绩构成
