# 数据采集

# 1、数据类型（按来源划分）

  • 商业数据
    • 是指一个产业，其价值链链熵各个重要环节的历史信息和即时信息的集合，其内容包括商业企业内部数据、分销渠道数据、消费市场数据等
  • 互联网数据
    • 是指互联网上所获得的数据的总称，包括图像文本视频网页等
  • 传感器数据
    • 由感知设备或传感设备感受、测量及传输的数据。感知设备或传感设备可以包括1个或多个传感器。传感数据种类有很多，如人身体的传感数据（如血压，血糖，心率等数据），网络信号的传感数据和气象的传感数据。传感数据可以用于数据分析。

# 2、数据类型（按类型分）

  • 结构化数据
    • 可直接以行、列形式进行储存和处理的数据，即数据库存储的数据。
  • 半结构数据
    • 不能被机器直接读取的数据，但是又存在一定的结构，例如百度百科中名人的履历。
  • 非结构化数据
    • 文本图像视频等

# 3、数据类型（按产生方式划分）

  • 线上行为数据
    • 人的行为所产生的数据。比如页面数据、交互数据表单数据、会话数据等。例如抖音人均刷抖音的时间，或者点赞数
  • 内容数据
    • 应用日志电子文档及其数据、语音数据、社交媒体数据等。

# 4、数据采集方式

  • 人工采集
    • 百度、谷歌、问卷调查
  • 网络数据自动采集
    • 网路爬虫或网络公开API，获取图片，音频，视频等
